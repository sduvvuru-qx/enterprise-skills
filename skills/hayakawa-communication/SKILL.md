# Hayakawa-san Communication

## Always load first

Read the working profile:

`/Volumes/☯Duality/spaces/professional/sduvvuru-qx/team-profiles/hayakawa-shinichiro.md`

If the user pasted a current Teams/email message, use that as the freshest evidence and adapt the profile guidance.

## Core posture

Treat Hayakawa-san as a **senior IPM alignment partner / routing gate**, not just a recipient.

Default tone: respectful, concise, operational, and route-aware.

## Strong default Teams pattern

```text
Hayakawa-san — quick alignment on [program/topic].

Purpose: [why this is needed]
Audience/use: [who will see/use it]

Current status:
- [fact]
- [open item]

Suggested next step:
- [recommended action]

Is this direction OK, or would you prefer I hold / route differently?
```

## For sensitive heat sink / GKN / supplier data

State purpose, recipient, and data type before sharing. Ask whether to hold for redaction/review.

## For JP DE follow-up

Avoid parallel directives. Ask whether to send one consolidated follow-up or hold.

## For decisions

Give 2–3 options and a recommendation. He has evidence of selecting a numbered option.

## For files

Mention intended audience and confirm access. If he reports access trouble, fix permission first and resend.

## Avoid

- Generic “any update?” pings.
- Escalation language without blocker/impact/date.
- Forwarding sensitive files without purpose/audience/routing confirmation.
- Long mixed-topic Teams messages.
