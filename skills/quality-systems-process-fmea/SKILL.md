# quality-systems-process-fmea

Use when the work shifts from design failure analysis into process failure analysis.

## When to Use

- PFMEA
- process risk analysis
- manufacturing-side failure review

## Workflow

1. Identify the process step.
2. Define process failure mode, effect, and cause.
3. Score and rank the process risk.
4. Link mitigations back to control and work-instruction artifacts.
